About Departureboard Admin

About

Departure Board Admin allows you to remotely control your boards from a different device. Once you register your board using the unique id you can change what content is being displayed from anywhere in the world on any device.

How to Use

  1. First you will need to login . To do this just click login at the top right of the page. This will need to be the same account as departure board admin.
  2. You should now go to https://admin.leddepartureboard.com and login using the same account.
  3. Then on departure board admin click the + button on the top right of the menu bar. Then click Add New Board.
  4. You should now give your board a name and choose a station and any other configuration. Once done click Save to add your new board, a message should pop up with a unique id (It is also shown on the boards page).
  5. Copy this id and then go to https://www.leddepartureboard.com/settings on the device you want to setup. Click on the Departure Admin section and paste the id into the Unique Id field, Then enable the Enable board slider and then click Save.
  6. As long as your logged in correctly then the board should instantly go to the page you configured in departure board admin.
  7. Now that you have setup the board you can visit the boards page in departureboard admin and click edit and mess around with the different configuration options.

Tips

  • You can visit https://admin.leddepartureboard.com on your phone and do all the same functions you could from your computer.
  • If you have multiple boards that you want to configure the same then Add a new group to have the configuration in one place.

Groups

  1. To add a new group click the + icon on departure board admin and click Add a new group
  2. Give the group a name and configure it how you want it. Then tick the boards you want to add to the group. Once done click save.
  3. You should now notice all the boards you added to the group should instantly change to the group config. The board config will be ignored when they are in a group.

You can remove a board from a group by editing the group and unticking the board. When you click save the board should revert back to the board config.
