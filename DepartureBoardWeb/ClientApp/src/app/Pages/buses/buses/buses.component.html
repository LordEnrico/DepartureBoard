Buses

About

You can view bus departure boards as found at bus stops in the uk. This is currently in beta phase and not complete. Beta

How to Use

  1. You will need to get the ATCO code of the bus stop. To do so head over to https://www.openstreetmap.org/. Click on a bus stop and it will show the code like so:


  2. One you have the ATCO code just copy it into the url https://leddepartureboard.com/buses/singleboard/InsertYourAtcoCodeHere

Example

Westminster Abbey bus stop https://leddepartureboard.com/buses/singleboard/490014495R
