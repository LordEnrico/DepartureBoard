Led Departure Board

Welcome

Led Departure Board is a webapp designed to re-create the platform signs you get at uk train stations. The data is collected live from national rail and displayed in a custom panel.

The main use case for this app is to set it up on a permanent screen which constantly shows the next trains at a specific station. The boards refresh every 16 seconds with the most up to date data, so there is no need to manually refresh.

How to Use

  1. To change the station just modify the url. E.g www.leddepartureboard.com/EUS will show euston station times
  2. To change the amount of boards on the screen just add a number to the end. E.g www.leddepartureboard.com/EUS/12
  3. To view a single platform style board just modify the url with a prefix of 'singleboard'. E.g www.leddepartureboard.com/singleboard/EUS

If you do not know the station code of your station then just google "station code for XXX"
