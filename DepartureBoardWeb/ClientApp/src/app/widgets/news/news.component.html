News

Settings Page

There is now a Settings page that allows you to set certain config. Please email me with more Setting suggestions

Search by Station Code

You can now also type station codes into the search to retrieve the station. E.g "RDG" will return Reading

Hide Menu

You can now hide the menu by adding the query param ?hideMenu=true e.g /EUS/5?hideMenu=true

Hide Clock

You can now hide the clock by adding the query param ?hideClock=true e.g /EUS/5?hideClock=true
